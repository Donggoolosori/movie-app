# Description

영화 정보 api를 사용하여 영화 리스트를 보여주는 app.

## 개발 언어

- React

## 주요 기능
- 영화 정보를 rank 순으로 보여준다.
- 포스터, 개봉일, 장르, 줄거리 정보
- 화면 크기에 따른 반응형 웹
